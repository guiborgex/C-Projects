# Soma dos Elementos de Cada Linha da Matriz

O programa lê dois valores inteiros m e n, depois lê uma matriz m×n contendo números reais.

Depois disso, o programa:

Calcula a soma dos elementos de cada linha

Armazena cada soma em um vetor

Exibe o vetor resultante, onde cada posição representa a soma de uma linha

É um exercício básico para praticar o uso de matrizes, vetores, laços aninhados e operações simples com números reais em C.

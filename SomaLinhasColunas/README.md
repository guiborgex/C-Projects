# Soma dos Elementos de Cada Linha da Matriz

O programa lê dois valores inteiros — o número de **linhas (m)** e **colunas (n)** — e depois lê uma matriz **m×n** contendo números reais.  
Depois de armazenar os dados, ele calcula e mostra:

- A **soma dos elementos de cada linha**  
- Cada soma armazenada em um **vetor**  
- O **vetor resultante**, onde cada posição representa a soma de uma linha da matriz  

É um exercício para praticar **matrizes**, **vetores**, laços aninhados e operações simples com números reais em C.

# Soma e Média de Valores em um Vetor

O programa lê um valor **N**, depois lê **N números reais** e armazena todos eles em um vetor.

Depois disso, o programa:

- Mostra **todos os valores** do vetor  
- Calcula e exibe a **soma** dos elementos  
- Calcula e exibe a **média** dos elementos  

É um exercício básico para praticar o uso de vetores, laços e operações simples com números reais.

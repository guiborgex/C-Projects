# Cálculo das Raízes de uma Equação do Segundo Grau

O programa lê os três coeficientes (**a**, **b** e **c**) de uma equação do segundo grau.  
Usando a fórmula de Bhaskara, ele calcula e mostra os valores das raízes **x1** e **x2**, ambos com **quatro casas decimais**.

Se o valor do discriminante (delta) for negativo, o programa informa que a equação não possui raízes reais.

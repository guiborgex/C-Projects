# Conversor de Temperaturas (Celsius ↔ Fahrenheit)

Este programa faz a conversão de temperaturas entre **Celsius** e **Fahrenheit**.  
O usuário escolhe qual tipo de conversão deseja fazer digitando **C** (para converter de Celsius para Fahrenheit) ou **F** (para converter de Fahrenheit para Celsius).

Depois de mostrar o resultado, o programa pergunta se o usuário quer repetir o processo.  
Ele continua rodando enquanto a resposta for **'s'** ou **'S'**.

É um exercício simples para praticar leitura de caracteres, uso de laços `do/while` e cálculos básicos de conversão de temperatura.

# Diagonal Principal e Contagem de Números Negativos

Este programa lê um valor inteiro **N** (máximo 10), que representa a dimensão de uma matriz **N x N** formada por números inteiros.

Depois de ler todos os elementos da matriz, o programa faz dois processos:

1. **Mostra a diagonal principal da matriz**  
2. **Conta quantos valores negativos existem na matriz**

É um exercício básico para praticar leitura de matriz, laços aninhados e análise de valores.

# Altura Média e Pessoas com Menos de 16 Anos

O programa lê o **nome**, a **idade** e a **altura** de **N pessoas**.  
Depois de armazenar os dados, ele calcula e mostra:

- A **altura média** do grupo  
- A **porcentagem** de pessoas com **menos de 16 anos**  
- Os **nomes** das pessoas com menos de 16 anos (se houver)

É um exercício para praticar vetores, estruturas simples de dados e cálculos usando informações armazenadas.
